先打开编辑器,引入脚本,并放入相关组件来看下数据连接是否正确。这个页面制作过程,我们用了2个组件库: bootstrap还有页面工具箱库
依次放入图片、上传按钮、文本、列表等页面组件。 对应连接关系如下： 

|组件|属性|配置|连接|
|--|--|--|--|
| 上传 | 已选 | -- |  文件地址  |
| 文本 | 内容 | -- |  古诗  |
| 文本 | 内容 | -- |  作者  |
| 文本 | 内容 | -- |  标题 |
| 图片 | 地址 | -- |  文件地址  |

|组件|交互|动作|
|--|--|--|
| 上传 | 文件改变 |文件选中 |

配置完成后运行查看效果。 可以看到基本功能正常

## 布局和美化 
现在我们来布局最终的输出页面。  
### 布局
首先，图片和诗词、颜色列表是层叠并且全屏的， 我们在页面放置一个“多层容器”，并勾选中“填满”选项

然后将图片和一个弹性容器放置其中。 图片在最底层展示内容，图片适应选拉伸

弹性容器按左右布局，再放入2个弹性容器， 
1、左边子容器按纵向、排列对齐两边、交叉对齐起点方式， 上下分别放入包含上传按钮的容器， 包含列表的容器 。 
2、右边子容器按横向布局，将古诗、作者及标题的文本放置其中

### 美化
1、对于古诗词及作者， 我们选择样式 样式工具箱 / 书写方向 / 垂直从右到左， 背景选择Bootstrap组件 / 背景黑，文字颜色白，内边距上间距2、下间距2，透明度75%。 字体上在全局输入子芒星字体地址，然后这些文字选择子芒星字体

2、上传部分，按钮和整体界面不协调。 但是按钮可以改变展示样式的。放置一个bootstrap图标组件，白色，选图片图标，然后将图标拖拽到上传组件中。

3、下方文字，调整其边距和大小

经过以上布局和美化的调整，页面达到了最终效果。
